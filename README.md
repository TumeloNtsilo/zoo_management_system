Zoo Management System
Overview

The Zoo Management System is a simple console-based Java application that allows users to manage animals in a zoo.
Users can add animals, view all animals, and search for animals by name using a menu-driven interface.

This project demonstrates object-oriented programming concepts and basic exception handling in Java.

Features
-  View all animals in the zoo
-  Add different animal species
-  Search for animals by name
-  Menu-based user interaction
-  Input validation using exception handling
  
Technologies Used
-  Java
-  Object-Oriented Programming (OOP)
-  Exception Handling
-  ArrayList collection
-  How to Run the Program
-  Open the project in your Java IDE
-  Run the ZoopApp.java file
-  Follow the on-screen menu options

Example menu:

1. View Animals
2. Add Animal
3. Search Animal
4. Exit
